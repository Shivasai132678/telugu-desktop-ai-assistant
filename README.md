📖 Overview
Bujji is a personal AI assistant that runs entirely on your local machine. It understands natural language voice and text commands, executes system actions, and speaks back in both English and Telugu — all without an internet connection.

Inspired by the AI companion from Kalki 2898 AD, Bujji is designed to be a real, always-listening desktop assistant for everyday use.

✨ Features
🎙️ Voice Input — Speak commands via microphone using OpenAI Whisper (offline STT)

🔊 Voice Output — Natural speech in English and Telugu using Piper TTS / XTTS-v2

🧠 Local LLM Brain — Powered by Gemma4:e2b or Qwen via Ollama (no API key needed)

🌐 Multilingual — Responds in Telugu and English; auto-detects script

💻 System Control — Open apps, websites, adjust volume, take screenshots via voice

😴 Wake Word Detection — Stays silent in the background; wakes up on trigger word

🖥️ Desktop GUI — Clean Flask + browser-based UI with chat interface and voice button

📦 Standalone App — Packaged as a native desktop app (PyWebView + PyInstaller)

🔌 OpenAI-Compatible API — /v1/chat/completions proxy layer for ecosystem integration

🔒 Fully Offline — No data sent to any external server
